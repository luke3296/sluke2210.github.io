Selection sort

  1. Find the minimum value in the list
  2. Swap it with the value in the first posisition
  3. repeat the steps above for the remainder of the list for for the next value

Insertion Sort

  1. Insert the first elelment into a new list. (a list of size 1 is allready sorted)
  2. Insert the second element into the allready sorted list
  3. For the theird element to the nth element repeat step 2

Formal disrciption

Given Array T[1..i-1]
for i:=2 to n
    temp := T[i]
    j:=i-1;
    while j>0 AND temp < T[j]
        T[j+1] := T[j]
        j := j-1
    T[j+1] := temp;

Not all cases are the same for insertion sort, because we can reach one of the exit conditions for the while loop, namely temp < T[j]
Fundermental operation

    while j>0 AND temp < T[j]

We will do Best case, e.g when the case is in sorted order. In the best case temp < T[j] will allways be false so we will allways skip the while loop. We end up with the sum of 1, from i=2 to n. Cos we starting at 2 we'll have one less 1 that n 1's so this is equal to n-1.
Best case O(n)

Insertion sort worst case:

The worst case is when the items are in descending order. For worst case

temp < T[j]

will allways be true, this means we get i fundermental operations per loop in i
